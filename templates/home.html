For Teachers

Create Test

Teachers can create new Tests. Currently MCQ questions with single option correct type can be added.

Authorize Students

Students can take the test only if the teacher permits them. Teacher can either allow or deny the student.

Add Batches

Teacher can allow complete batches of students to take test at Once. Thus all the students of a class can be added at once.

For Students

Search Test

Students can search for test from available list of tests and then send a request to join the test.

Instant Score

Once a student attempts the test he/she can immediately see their scores after the test. This is done at the backend.

Previous history

A student can view all the tests he/she has attempted and thus keeps a history of their performance which can help in improvment.
